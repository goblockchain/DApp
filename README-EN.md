### Local installation

To navigate in the project on your computer, execute the following commands on the root folder 'DApp'.

> Requirements
>> Necessary install node.js, [download here](https://nodejs.org/en/download/).


```
- npm install
- npm install -g bower
- bower install
- gulp dev
```
